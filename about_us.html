About Us

Project Diabipal focused on managing Diabetes using machine learning techniques and optical character recognition. In this work, Random Forest, Logistic Regression classification machine learning algorithms are used and evaluated on the datasets to find the prediction of diabetes and coronary diseases. Image processing algorithms and Tesseract Optical Character Recognition (OCR) engines are used to extract data from medical reports. A knowledge graph was used to improve the accuracy of the data extracted. Natural Language Processing (NLP) is used to answer the patient's questions and connect with the doctor. Also, topic modeling is used to suggest the best doctor based on experience. This project was nominated to NBQSA 2020 from Sri Lanka Institute of Information Technology

DiabiPal

DiabiPal is a mobile application catered for the needs of the diabetic patients.

Application Features

  • Scan medical reports using OCR
  • Predict Pre-diabetes, Diabetes Mellitus status
  • Predict the risk of cardiovascular diseases
  • Platform for communication with physicians
